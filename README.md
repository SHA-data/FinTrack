# 💰 FinTrack: AI-Powered Personal Finance Tracker

FinTrack is a comprehensive, modern personal finance management application built with **Flask**, **PostgreSQL (via Supabase)**, and **Groq AI**. It empowers users to take control of their financial life through intuitive transaction tracking, proactive budgeting, and personalized AI-driven financial insights.

## 🚀 Key Features

- **🔐 Secure Authentication**: Robust user signup and login system with encrypted passwords using Werkzeug.
- **📊 Intuitive Dashboard**: A high-level overview of your monthly income, expenses, and net balance with visual progress indicators.
- **💸 Transaction Management**: Easily record, categorize, and manage your daily transactions.
- **🎯 Smart Budgeting**: Set monthly spending limits for various categories (Food, Transport, Utilities, etc.) and track your progress in real-time.
- **⚠️ Proactive Budget Alerts**: Automated system triggers alerts when you approach or exceed your set budget limits, powered by database triggers.
- **🤖 AI Financial Insights**: Get personalized, actionable advice based on your spending patterns, generated by **Groq's LLaMA-3.3-70B** model.
- **📱 Responsive Design**: A premium, mobile-friendly UI built with vanilla CSS and modern glassmorphism aesthetics.

## ⚙️ Setup & Installation

### 1. Prerequisites
- Python 3.12 or higher
- [uv](https://github.com/astral-sh/uv) (recommended) or `pip`

### 2. Clone the Repository
```bash
git clone https://github.com/yourusername/fintrack.git
cd fintrack
```

### 3. Install Dependencies
Using **uv**:
```bash
uv sync
```
Using **pip**:
```bash
pip install -r finance_tracker/requirements.txt
```

### 4. Configuration
Create a `.env` file in the `finance_tracker/` directory (or use the existing one as a template):
```env
SUPABASE_URL = "your-supabase-url"
SUPABASE_KEY = "your-supabase-anon-key"
GROQ_API_KEY = "your-groq-api-key"
SECRET_KEY   = "your-flask-secret-key"
FLASK_ENV    = "development"
FLASK_DEBUG  = 1
```

### 5. Database Setup
The database schema and initial data are located in `finance_tracker/db/`.
1. Run `schema.sql` in your Supabase SQL Editor to create tables, views, and triggers.
2. Run `seed.sql` to populate default categories.

### 6. Run the Application
```bash
python finance_tracker/app.py
```
Visit `http://127.0.0.1:5000` in your browser.

## 📊 Database Schema Highlights

The project leverages advanced PostgreSQL features to ensure data integrity and automate business logic:

- **Triggers**: `trg_check_budget` automatically logs alerts when a transaction exceeds a budget limit.
- **Views**: `monthly_summary` and `category_totals` provide optimized data for dashboard charts and reports.
- **Stored Procedures**: Custom PL/pgSQL functions for atomic transaction and budget operations.
- **Cascading Deletes**: Ensures user data is cleaned up properly upon account deletion.
